A diferença entre Clients e Servers

(Voltar)

O Servidor é o computador no qual foi configurado em uma rede para oferecer algum tipo de serviço. O Cliente, é o computador que está do outro lado, que foi configurado para utilizar o serviço que é oferecido pelo servidor.
Neste caso, podemos afirmar então que o computador servidor é aquele que oferece um serviço e o computador cliente é aquele que se ocupa de um serviço.

Por exemplo, ao acessarmos uma página da web, a mesma é disponibilizada a partir de um computador servidor. O nosso computador, chamado de cliente, irá realizar as solicitações de acesso às páginas da web que estão armazenadas em um servidor.

Como é uma estrutura básica do HTML?

(Voltar)

A estrutura básica de um código HTML tem como foco formar um alicerce para a construção de uma página web.

O DOCTYPE html, diz para os navegadores que pretendemos escrever uma pagina em HTML de acordo com as especificações, sendo que o DOCTYPE não é uma tag do HTML, mas sim um tipo de palavra-chave.

A tag html, é o elemento onde toda a pagina será desenvolvida, sendo que ela recebe um atributo chamado lang="pt-br" que defini o idioma principal da pagina.

A tag head, é o elemento que compõe a cabeça da página, ela não será renderizada na tela pelo navegador, pois tem como papel principal ser uma tag de metadados.

A tag title, diz respeito ao título da página, o nome que aparece da página do navegador.

A tag que aparece na base de uma página HTML é a body que é o corpo da página, tudo que for escrito dentro dessa tag aparecerá no navegador, portanto é onde vai todo o conteúdo da página.

Utilizamos a tag P, para representar um paragráfo.

Os elementos h1,h2, h3, h4, h5,e h6, representam os seis niveis de titulos, do mais alto para o mais baixo.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados!

A lista ordenada enumera os elementos

Dessa maneira:

  1. primeiro item
  2. segundo item
  3. terceiro item


A lista não-ordenada coloca marcadores antes de cada elemento

Exemplo:

  • primeiro item
  • segundo item
  • terceiro item
